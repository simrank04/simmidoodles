AAA Environmental Services Inc

Project Type: Static Responsive Website.

Role: UI/UX Designer, UX Researcher, Logo Design & Front-End Developer.

Link to the Website: AAA Environmental Services Inc

Project Overview


This project involved the end-to-end design and front-end development of a static, responsive website for AAA Environmental Services Inc., a company specializing in professional cleanup and remediation services. The goal was to create a modern, clean, and user-friendly website that effectively communicates the company’s services, builds trust, and enhances online presence.


Tools & Tech Used


  • Design & Research: Figma, Adobe Illustration, Procreate, Canva.
  • Development : HTML5, CSS3, JavaScript, Bootstrap.
  • Code Editor : Visual Studio Code.

My Contribution


  • Conducted user research and competitor analysis to identify target user needs and industry trends.
  • Created low-fidelity and high-fidelity wireframes in Figma, establishing a clear user flow and intuitive layout.
  • Designed the complete UI from scratch, aligning with the brand's visual identity and service ethos.
  • Developed the responsive front-end using clean HTML, CSS, JavaScript, and Bootstrap, ensuring cross-device compatibility.
  • Designed a new logo and accompanying brand assets, including color palette, typography, and iconography.
