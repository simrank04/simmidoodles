Bitebook - Your Recipe Friend.

Project Type: Responsive Mobile Application.

Role: UI/UX Designer, UX Researcher & Logo Design.

Link to the Website: Bitebook Applicaiton.

Project Overview


BiteBook is a mobile application that allows users to discover delicious recipes with filtering different Cuisines, Cooking Time, Reviews. BiteBook stand out among the competitors as it has Voice & Scan and search feature.


  • User can Scan & voice search.
  • User can Watch recipe videos.
  • User can filter by cooking time.
  • User can Share recipe link with family/friends/social media.


    • Tools & Tech Used
