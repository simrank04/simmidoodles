Buyerfolio.


Project Type: Static Responsive Website & Mobile Application.

Role: UI/UX Designer & UX Researcher.

Link to the Website: Buyerfolio

Project Overview


Buyerfolio is a web application platform designed to simplify and democratize shared homeownership. It evaluates users’ financial readiness through its AI-powered Folio Score®, helps match co-buyers based on financial compatibility, and supports shared mortgage management.
The goal: reduce friction for people aiming to co-buy property, give lenders transparency, and give co-buyers confidence in decision-making.


My Contribution

  • User research and competitor analysis.
  • Designed a new logo and accompanying brand assets, including color palette, typography, and iconography.
  • Designed the complete UI from scratch, aligning with the brand's visual identity and service ethos.
  • Creating wireframes, prototypes, and final UI screens in Figma.
  • Full responsive Front-End Designing.

Key Contributions & Challenges


  • Simplified Folio Score® input so users understand what data is needed and why.
  • Designed an intuitive co-buyer matching UI that visualizes compatibility (financial, goals, preferences).
  • Established clear dashboard experience showing ownership breakdown, payments, alerts.
  • Balanced complex financial data with a clean, user-friendly interface.
  • Managed design handoff to developers, ensuring assets and specs were clear,
