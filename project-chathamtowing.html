Chatham Towing.


Project Type: Static Responsive Website.

Role: UI/UX Designer, UX Researcher, Logo Design & Front-End Developer.

Link to the Website: Chatham Towing.


Project Overview


Chatham Towing is a professional towing service website designed to build credibility, highlight services, and allow potential customers to easily get in touch. The goal was to create a clean, responsive, and user-friendly website that reflects trustworthiness and professionalism for a local business.


My Role


  • UI/UX Designer → Created the entire website layout and brand identity using Figma and Canva.
  • Front-End Developer → Hand-coded the website using HTML5 and CSS3, ensuring responsiveness across devices.

My Contribution


  • Delivered a complete site from design → prototype → live build.
  • Designed a new logo and accompanying brand assets, including color palette, typography, and iconography.
  • Designed the complete UI from scratch, aligning with the brand's visual identity and service ethos.
  • Low-Fi & High-Fi Wireframing and UI design in Figma from scratch.
  • Focused on UX best practices: clear call-to-actions, easy navigation, prominent contact info.
  • Logo and Brand asset creation.
