FitnestX - Everybody can Train.

Project Type: Responsive Mobile Application.

Role: UI/UX Designer, UX Researcher & Logo Design.

Link to the Application: FitnestX.


Project Overview


FitnestX is a Application that combine workout app, meal tracker app and health app in one so that users will no longer need to use separate apps to keep.
Designed by Simran Kaur & was Designed as a Freelancing project.


  • User can Set their Goals.
  • User can see Workout Tracker & Schedule Workout.
  • User can filter by type of Training.
  • User can Meal Plan, Track Sleep & Progress.
